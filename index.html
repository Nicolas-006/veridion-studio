Preguntas Frecuentes

¿En que tipo de proyectos trabaja Veridion Studio? +

Nos especializamos en Landing Pages estratégicas orientadas a conversión.
También realizamos rediseños web, mantenimiento mensual y gestión de dominio y hosting.

¿Cómo es el proceso de trabajo? +

Nuestro proceso es claro y transparente:
1. El cliente completa un formulario con la información del proyecto.
2. Analizamos el pedido y enviamos un presupuesto detallado junto con el acuerdo de servicio.
3. Una vez aceptado el presupuesto y confirmado el acuerdo, se abona el 50% inicial para comenzar el proyecto.
4. Presentamos una propuesta estructural inicial (wireframe) basada en los objetivos definidos. Incluye hasta 2 rondas de ajustes antes del desarrollo final.
5. Tras la aprobación del diseño, desarrollamos la landing page.
6. Se realizan ajustes finales dentro del alcance acordado y se abona el 50% restante.
7. Entregamos la página web lista para su publicación o la publicamos si el servicio de hosting fue contratado con nosotros.

¿Puedo solicitar cambios durante el proyecto? +

Sí.
El proyecto incluye instancias de revisión en cada etapa para asegurar que el resultado esté alineado con tus objetivos.
En la fase de estructura (wireframe) se incluyen hasta 2 rondas de ajustes, donde podemos reorganizar secciones o mejorar el enfoque estratégico.
Durante el desarrollo visual se incluye 1 ronda de ajustes finales, orientada a detalles de diseño, textos o elementos visuales.
Si se requieren cambios adicionales o modificaciones estructurales fuera del alcance acordado, podrán presupuestarse aparte antes de realizarse.

¿Cuánto tiempo tarda el desarrollo de una página web? +

El desarrollo de una Landing Page suele demorar entre 5 y 10 días hábiles desde la recepción del anticipo y el contenido completo.
Los tiempos pueden variar según la complejidad del diseño y la rapidez en la aprobación de los contenidos.

¿Cuánto cuesta una página web? +

Cada proyecto es diferente, por lo que no trabajamos con precios fijos.
El presupuesto depende del tipo de web, el nivel de diseño, las funcionalidades solicitadas y los servicios adicionales.
Luego de analizar tu proyecto, te enviamos un presupuesto claro y personalizado.

¿Cómo se puede pagar el servicio? +

Aceptamos pagos mediante:
- Efectivo
- Transferencia bancaria
Próximamente se incorporarán pagos con tarjeta de crédito y débito.

¿Es necesario pagar todo por adelantado? +

No.
Trabajamos con un esquema de pago profesional:
- 50% al iniciar el proyecto
- 50% al finalizar, antes de la entrega final
Esto asegura un compromiso mutuo y transparencia para ambas partes.

¿El servicio incluye dominio y hosting? +

Podemos gestionar el dominio y el hosting por vos, facilitando todo el proceso técnico.
Este servicio puede incluirse dentro del proyecto o contratarse aparte.

¿La página web será compatible con celulares? +

Sí.
Todos nuestros sitios web son responsive, adaptándose correctamente a celulares, tablets y computadoras.

¿Pueden integrar WhatsApp y redes sociales? +

Sí.
Podemos integrar botones de WhatsApp, enclaces a redes sociales y otros medios de contacto para facilitar la comunicación con tus clientes.

¿Ofrecen mantenimiento web? +

Sí.
Todas nuestras Landing Pages incluyen un período de garantía de 30 días posteriores a la entrega.

Durante este período corregimos sin costo cualquier error técnico o fallo de funcionamiento relacionado con el desarrollo realizado por nosotros, asegurando que la página opere correctamente según lo acordado.
La garantía no cubre cambios de contenido, modificaciones estructurales solicitadas posteriormente ni intervenciones realizadas por terceros.

Una vez finalizado el período de garantía, ofrecemos un servicio de mantenimiento mensual opcional que puede incluir actualizaciones, pequeños ajustes de contenido y soporte técnico para garantizar el correcto funcionamiento del sitio.
En caso de no contratar mantenimiento mensual, también es posible solicitar modificaciones puntuales cuando se necesiten, las cuales se presupuestan de forma individual según el alcance del cambio

¿Pueden rediseñar una web que ya tengo? +

Sí.
Realizamos rediseños completos o parciales para modernizar tu sitio, mejorar la experiencia del usuario y optimizarlo para dispositivos móviles.

¿Mi página web podrá ampliarse en el futuro? +

Si en el futuro necesitás ampliar tu presencia digital, podemos escalar tu proyecto hacia nuevas secciones o funcionalidades según tus necesidades.
